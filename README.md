# KT GenieLabs Dev-Challenge 2022

## 개요
* 본 대회는 KT GenieLabs가 주최한 대회입니다.
* 음식 이미지 분류 모델을 설계하는 것이 본 대회의 목표였습니다.
* 팀명은 ISTHISFOOD였으며, TOP 10에 선발되어 본선에 진출하였습니다.
